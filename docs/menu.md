## 菜单
左侧导航栏菜单都在此模块管理。菜单和用户权限绑定。
![menu](http://pic.yupoo.com/craber_v/4fb3c761/e0c2788e.jpeg)

### 添加菜单

![menu](http://pic.yupoo.com/craber_v/4fb3c761/e0c2788e.jpeg)

表单字段说明：
- `name` 模块名称
- `route` 模块url pathname，按照 dva 路由格式添加，具体参照 dva 文档
- `icon` 菜单名称前的小图标
- `status` 是否启用
- `menu pid` 左侧导航栏父级菜单 id
- `breadcrum pid` 面包屑上一级 ID
- `bind apis` 该模块页面允许访问的 api path
